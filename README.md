# engineeing-manage-system

智能化工程管理系统：项目合同管理、报价管理、项目进度管理、项目维保（含多点位报价、季度结算、验收结算）。

## 技术栈

- 后端：Java 17 + Spring Boot 3.2 + MyBatis Plus 3.5 + MySQL 8 + Redis 7 + JWT
- 前端：Vue 3 + Element Plus + Chart.js（Linear/Notion 风格 UI）

## 目录

- `docs/superpowers/specs/` 设计规格
- `docs/superpowers/plans/` 实现计划
- `需求文档.md` 原始需求
