## 一、设计风格定位

- **风格**: Linear/Notion 风(现代极简、克制配色、细腻阴影、紧凑高效)
- **原因**: B 端管理系统的本质是信息密度高、操作频繁,Apple 官网风(大留白、低密度)与之冲突。Linear/Notion 风在审美与效率间取得最佳平衡
- **参考**: Linear (https://linear.app), Notion (https://notion.so)

## 六、交互与动效

### 6.1 动效强度: 轻微动效
| 场景 | 动效 |
|------|------|
| 全局过渡 | 150-200ms ease |
| 弹窗 | 淡入 + 轻微上移 (4px) |
| 抽屉 | 滑入 (translateX) |
| Hover | 背景色渐变 |
| Focus | 主色描边 + 轻微发光 |
| 加载 | 骨架屏优先,避免白屏 |

### 6.2 不做的事
- 不用缩放动画
- 不用爆炸/弹跳效果
- 不用全屏过渡

## 七、组件风格

### 7.1 按钮
- 主按钮: 实心填充主色 `#4F6BED`,白字
- 次按钮: 白底 + 1px 主色边框,主色字
- 文字按钮: 无背景,主色字,hover 浅填充
- 危险按钮: 实心填充 `#EF4444`
- 圆角 6px,padding 8px 16px

### 7.2 表格
- 去掉斑马纹
- 行 hover: 背景 `#F9FAFB`
- 行高 56px(舒适密度)
- 分隔线: 1px `#F3F4F6`(极淡)
- 表头: 背景 `#FAFAFA`,字色 `#4B5563`,Medium 字重
- 交互模式: 查看为主,点击行打开右侧详情抽屉(非行内编辑)

### 7.3 表单
- 输入框: 白底 + 1px `#E5E7EB` 边框,圆角 6px
- 聚焦: 1px 主色边框 + `0 0 0 3px rgba(79,107,237,0.1)` 发光
- 标签: 14px Regular,`#4B5563`,置于输入框上方
- 字段间距: 20px

### 7.4 标签(Tag)
- 轻填充风格: 主色 10% 透明背景 + 主色文字
- 例: 背景 `rgba(79,107,237,0.1)`,字 `#4F6BED`
- 圆角 4px,padding 2px 8px

### 7.5 导航(侧栏)
- 背景 `#FAFAFA`
- 选中项: 主色 10% 透明背景 + 主色文字
- Hover: 背景 `#F3F4F6`
- 扁平化,无 3D 效果

### 7.6 卡片
- 白底 + 1px `#E5E7EB` 边框
- 圆角 8px
- 内边距 24px
- 标题: 16px Semibold,正文 14px Regular

### 7.7 弹窗/抽屉
- 弹窗: 白底,圆角 12px,阴影 `0 8px 24px rgba(0,0,0,0.12)`
- 抽屉: 从右侧滑入,宽度 480px(标准)/ 640px(宽)
- 遮罩: `rgba(0,0,0,0.4)`

## 八、技术选型

| 项 | 选型 | 理由 |
|----|------|------|
| UI 组件库 | Element Plus + 重写主题 | 生态成熟,通过 SCSS 变量覆盖调出 Linear 风 |
| 图表库 | Chart.js | 轻量,默认风格现代 |
| 图标 | Lucide Icons | 线条简洁,符合 Linear 风 |
| 字体 | Inter + PingFang SC | 现代无衬线,中英文协调 |
| 动效 | 内置 CSS transition + Element Plus 过渡 | 轻量,无需引入额外动画库 |

## 九、Element Plus 主题定制要点

通过 SCSS 变量覆盖 Element Plus 默认主题:

```scss
// 主色
$--color-primary: #4F6BED;

// 文字色
$--color-text-primary: #1F2937;
$--color-text-regular: #4B5563;
$--color-text-secondary: #9CA3AF;

// 边框
$--border-color-base: #E5E7EB;
$--border-color-light: #F3F4F6;

// 圆角
$--border-radius-base: 6px;
$--border-radius-small: 4px;

// 字体
$--font-size-base: 14px;
$--font-family: 'Inter', 'PingFang SC', sans-serif;
```
